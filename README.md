# Music Genre Classification with CNNs and Audio-Based Song Recommendation

## Project goal
This project explores music genre classification using convolutional neural networks on Mel-spectrograms and extends the approach with a simple audio-based song recommendation system.

## Planned structure
- `data/` raw and processed audio data
- `notebooks/` experiments and weekly progress
- `src/` reusable Python code
- `results/` plots and trained models

## Main tasks
1. Audio loading and visualization
2. Spectrogram and Mel-spectrogram generation
3. Baseline model
4. CNN-based genre classification
5. Audio-based recommendation
